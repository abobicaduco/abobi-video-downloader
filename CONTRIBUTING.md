# Contributing — Abobi Video Downloader

Fork mantido por [@abobicaduco](https://github.com/abobicaduco). Baseado no [Seal](https://github.com/JunkFood02/Seal) (GPL-3.0).

## Bugs e ideias

Abra uma [issue](https://github.com/abobicaduco/abobi-video-downloader/issues) com descrição clara, link do vídeo e passos para reproduzir.

## Build

```bash
./gradlew assembleDebug --no-daemon
```

APK release: Actions → **Build Release APK** (branch `caduco-stable`).

## Traduções

Strings em `app/src/main/res/values-*/strings.xml`. Pull requests bem-vindos.
